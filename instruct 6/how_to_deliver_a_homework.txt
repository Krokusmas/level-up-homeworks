Как сделать первое ДЗ
у вас уже будет готовая папка public/views/css/homework-1
и в ней уже есть index.html
и есть style.css в папке css

вот просто там и пишите

чтобы видеть результат, открыть Git Bash из папки level_up_homeworks
запустить node server.js

открыть в браузере http://localhost:8000/css/homework-1/

когда вы закончили ДЗ
останавливаем сервер
пишем в Git Bash без кавычек
'git add .'
'git commit -m "CSS HW 1"' - тут нужны двойные кавычки вокруг CSS HW 1
'git push origin master'

когда вы зайдете на свой репозиторий, вы должы будете увидеть изменения в папке  public/views/css/homework-1
на сайте там можно перемещаться как в файловой системе


Как делать последующие ДЗ
создаём папку в разделе css - homework-n
где n - порядковый номер занятия соответствующего раздела
т.е. homework-1; homework-2; homework-3 и т.д.
внутри такой папки всегда должен быть index.html и папка css. Внутри папки css должен быть style.css. Если есть картинки, создавайте под них папку images на одном уровене с папкой css.

После того, как всё готово пишите всегда три команды без кавычек
'git add .'
'git commit -m "CSS HW <N>"' - тут нужны двойные кавычки вокруг CSS HW <N>. <N> заменяете на порядковый номер занятия соответствующего раздела
'git push origin master'
